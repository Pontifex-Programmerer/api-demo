# Book Library Frontend

This is the frontend web interface for the book library application.

## Installation

```bash
cd frontend
npm install
```

## Running

First, make sure the backend server is running on port 3000, then:

```bash
npm start
```

The frontend will run on http://localhost:3001

## Features

- View all books
- Search books by title, author, genre, or year
- Responsive web interface
- Clean MVC architecture

## Dependencies

This frontend requires the backend API to be running. Make sure to start the backend server first.
